# HTTP là gì

## HTTP
- Là một **Applications protocol** dạng **stateless**
- Tương tác **client-server** theo **HTTP** thì phần **server** sẽ không lưu dữ liệu trên **client**

- Có 4 cách lưu **data** của **client** khi xây dựng **Web Application** bao gồm :
    - **URL Rewriter**
    - **Form**
    - **Cookie**
    - **HTTP Session**
